# 部署运维文档

## 环境要求

### 基础环境

| 组件 | 最低版本 | 推荐版本 | 说明 |
|------|----------|----------|------|
| JDK | 17 | 21 LTS | 运行时环境 |
| Maven | 3.8 | 3.9+ | 构建工具 |
| PostgreSQL | 15 | 16 | 主数据库 (需 pgvector) |
| Redis | 7.0 | 7.2+ | 缓存 |
| Kafka | 3.0 | 3.6+ | 消息队列 |

### 硬件建议

| 环境 | CPU | 内存 | 磁盘 |
|------|-----|------|------|
| 开发环境 | 2 核 | 4 GB | 20 GB |
| 测试环境 | 4 核 | 8 GB | 50 GB |
| 生产环境 | 8 核+ | 16 GB+ | 100 GB+ SSD |

## 本地开发部署

### 1. 克隆项目

```bash
git clone <repository-url>
cd education-agent
```

### 2. 启动依赖服务

使用 Docker Compose 一键启动：

```bash
docker-compose up -d postgres redis kafka
```

或手动启动各服务。

### 3. 初始化数据库

```bash
# 创建数据库
psql -U postgres -h localhost -c "CREATE DATABASE openagent_education_agent;"

# 启用 pgvector 扩展
psql -U postgres -h localhost -d openagent_education_agent -c "CREATE EXTENSION IF NOT EXISTS vector;"

# 执行初始化脚本
psql -U postgres -h localhost -f 02-scripts/db-init-education-agent.sql
```

### 4. 配置环境变量

```bash
# AI 模型 API Key
export OPENAI_API_KEY=your-api-key

# 可选: 覆盖默认配置
export SPRING_DATASOURCE_URL=jdbc:postgresql://localhost:5432/openagent_education_agent
export SPRING_DATASOURCE_USERNAME=postgres
export SPRING_DATASOURCE_PASSWORD=your-password
export SPRING_DATA_REDIS_HOST=localhost
export SPRING_KAFKA_BOOTSTRAP_SERVERS=localhost:9092

# 多模型配置 (规划)
export AI_MODEL_PRIMARY=gpt-4o-mini
export AI_MODEL_ADVANCED=gpt-4o
export AI_MODEL_ROUTING_ENABLED=true
```

### 5. 构建与运行

```bash
# 编译打包
mvn clean package -DskipTests

# 运行 JAR
java -jar target/education-agent-1.0.0-SNAPSHOT.jar

# 或直接运行
mvn spring-boot:run
```

### 6. 验证部署

```bash
curl http://localhost:8202/education-agent/system/health
```

预期返回：

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "status": "UP",
    "agent": "教育培训智能体",
    ...
  }
}
```

## 生产环境部署

### JVM 参数建议

```bash
java -jar education-agent.jar \
  -Xms2g -Xmx4g \
  -XX:+UseG1GC \
  -XX:MaxGCPauseMillis=200 \
  -XX:+HeapDumpOnOutOfMemoryError \
  -XX:HeapDumpPath=/var/log/education-agent/heapdump.hprof \
  -Dspring.profiles.active=prod
```

### 多环境配置

通过 `spring.profiles.active` 切换配置：

| Profile | 说明 |
|---------|------|
| default | 本地开发 (application.yml) |
| dev | 开发环境 |
| test | 测试环境 |
| prod | 生产环境 |

## 故障排查

### 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 数据库连接失败 | PostgreSQL 未启动或配置错误 | 检查数据库连接参数，确认服务可达 |
| Redis 连接超时 | Redis 未启动或密码错误 | 检查 Redis 配置，确认 network 可达 |
| Kafka 消费异常 | Kafka broker 不可用 | 检查 bootstrap-servers 配置 |
| 模型调用失败 | API Key 无效或网络问题 | 验证 OPENAI_API_KEY 环境变量 |
| 端口冲突 | 8202 端口被占用 | 修改 `server.port` 配置 |
| pgvector 扩展不可用 | 未安装 pgvector | 执行 `CREATE EXTENSION vector;` |
| 向量检索失败 | 向量维度不匹配 | 确认 embedding 模型输出维度与配置一致 |
| 内存溢出 | 向量数据量过大 | 增加 JVM 堆内存，考虑分页加载 |

### 查看日志

```bash
# 查看应用日志
tail -f /var/log/education-agent/application.log

# 查看错误日志
grep ERROR /var/log/education-agent/application.log | tail -50
```
