# 应用设计文档

## 一、应用概述

Education Agent 是 OpenAgent 平台的子应用，作为独立可部署的 Spring Boot 服务运行，通过 REST API 对外提供智能教育服务。

### 基本信息

| 项目 | 值 |
|------|-----|
| 应用名称 | education-agent |
| 应用标识 | `education-agent` |
| 服务端口 | 8202 |
| 上下文路径 | `/education-agent` |
| 父工程 | openagent (1.0.0-SNAPSHOT) |
| 核心依赖 | agent-core (1.0.0-SNAPSHOT) |

## 五、异常处理设计

### 5.1 异常处理链

```
业务代码抛出异常
    ↓
GlobalExceptionHandler 捕获
    ↓
┌─────────────────────────────────────────────┐
│ BusinessException     → 200 + 业务错误码     │
│ ValidationException   → 400 + 参数错误信息   │
│ BindException         → 400 + 绑定错误信息   │
│ Exception (其他)      → 500 + 服务器内部错误  │
└─────────────────────────────────────────────┘
    ↓
统一 Result<T> JSON 响应
```

### 5.2 错误码体系

| 范围 | 类型 | 说明 |
|------|------|------|
| 200 | 成功 | 操作成功 |
| 400-499 | 客户端错误 | 参数错误、未授权、权限不足 |
| 500-599 | 服务端错误 | 内部错误、服务不可用 |
| 1001-1099 | 业务错误 | 智能体相关 |
| 2001-2099 | 业务错误 | 用户/认证相关 |
| 3001-3099 | 业务错误 | 课程/知识点相关 |
| 4001-4099 | 业务错误 | 辅导/对话相关 |
| 5001-5099 | 业务错误 | 学习计划相关 |
| 6001-6099 | 业务错误 | 内容生成相关 |
| 7001-7099 | 业务错误 | 教师助手相关 |
| 8001-8099 | 业务错误 | 作业/考试相关 |
| 9001-9099 | 业务错误 | 游戏化/积分相关 |

## 六、缓存设计

### 6.1 Redis Key 规范

```
education:{module}:{business}:{id}
```

| Key 模式 | 用途 | 过期时间 |
|----------|------|----------|
| `education:session:{sessionId}` | 对话会话上下文 | 30min |
| `education:user:profile:{userId}` | 用户画像缓存 | 1h |
| `education:user:stage:{userId}` | 用户学段信息 | 1h |
| `education:course:detail:{id}` | 课程详情缓存 | 30min |
| `education:knowledge:list:{courseId}` | 知识点列表缓存 | 1h |
| `education:knowledge:hot:{stageId}` | 热点知识点向量 | 2h |
| `education:question:bank:{subject}:{grade}` | 试题库缓存 | 1h |
| `education:plan:daily:{userId}:{date}` | 每日学习内容 | 当天 |
| `education:gamification:points:{userId}` | 用户积分 | 不过期 |
| `education:gamification:rank:{scope}` | 排行榜缓存 | 5min |
| `education:parent:report:{childId}:{date}` | 家长学情报告 | 1h |
| `education:agent:status` | 智能体状态 | 不过期 |
| `education:system:announcement` | 系统公告 | 10min |

### 6.2 缓存策略

- **Cache-Aside**: 先查缓存，未命中查 DB 并回填
- **写入时失效**: 数据更新时删除对应缓存
- **防穿透**: 空值也缓存（短过期时间）
- **防雪崩**: 过期时间加随机偏移量

registry.addMapping("/**")
    .allowedOriginPatterns("*")

## 九、定时任务

应用已启用 `@EnableScheduling`，支持定时任务：

```java
@Scheduled(cron = "0 0 2 * * ?")  // 每天凌晨2点
public void cleanExpiredSessions() {
    // 清理过期会话
}
```

### 规划中的定时任务

| 任务 | 周期 | 说明 |
|------|------|------|
| 过期会话清理 | 每天凌晨 | 清理过期的对话会话和 Redis 缓存 |
| 缓存预热 | 每小时 | 预热热点课程/知识点/排行榜缓存 |
| 学习统计聚合 | 每天凌晨 | 聚合前一日全学段学习数据 |
| 家长学情报告生成 | 每天晚上 | 生成小学生/初中生的学情日报 |
| 学生预警扫描 | 每天凌晨 | 扫描并识别可能掉队的学生 |
| 排行榜更新 | 每小时 | 更新班级/年级/全平台学习排行榜 |
| 积分过期清理 | 每月初 | 清理过期积分，生成积分月报 |
| 知识图谱增量更新 | 每天凌晨 | 处理新增/变更的知识点向量化 |
