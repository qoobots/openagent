# 数据库设计文档

## 基础信息

| 项目 | 值 |
|------|-----|
| 数据库类型 | PostgreSQL 15+ |
| 数据库名 | `openagent_education_agent` |
| 向量扩展 | pgvector |
| ORM 框架 | MyBatis-Plus |
| 字符集 | UTF-8 |

## 设计规范

### 命名规范

- 表名: 小写 + 下划线，前缀 `edu_`，如 `edu_course`
- 字段名: 小写 + 下划线，如 `course_name`
- 索引名: `idx_表名_字段名`
- 唯一索引: `uk_表名_字段名`

### 通用字段

所有业务表包含以下通用字段（继承自 `BaseEntity`）：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键，雪花算法生成 (ASSIGN_ID) |
| create_time | TIMESTAMP | 创建时间，自动填充 |
| update_time | TIMESTAMP | 更新时间，自动填充 |
| deleted | INT | 逻辑删除标记 (0=未删除, 1=已删除) |

### 主键策略

采用 MyBatis-Plus 的 `ASSIGN_ID` 策略（雪花算法），生成 19 位 Long 型唯一 ID。

---

## 实体基类

```java
@Data
public abstract class BaseEntity implements Serializable {
    @TableId(type = IdType.ASSIGN_ID)
    private Long id;

    @TableField(fill = FieldFill.INSERT)
    private LocalDateTime createTime;

    @TableField(fill = FieldFill.INSERT_UPDATE)
    private LocalDateTime updateTime;

    @TableLogic
    private Integer deleted;
}
```

## 向量检索配置

| 配置项 | 值 | 说明 |
|--------|-----|------|
| dimension | 1536 | 向量维度 (text-embedding-3-small) |
| index-type | ivfflat | 索引类型 |
| metric-type | cosine | 距离度量 (余弦相似度) |
| index-lists | 100 | IVFFlat 列表数 |

### 向量索引创建示例

```sql
-- 创建向量索引
CREATE INDEX idx_knowledge_embedding 
ON edu_knowledge 
USING ivfflat (embedding vector_cosine_ops) 
WITH (lists = 100);
```
