## 三、智能辅导模块

### 3.1 智能对话辅导 📋 P0

| 项目 | 说明 |
|------|------|
| 功能描述 | 基于 AI 的多轮对话辅导，支持上下文记忆、知识点关联问答 |
| 计划接口 | `POST /api/tutor/chat` |
| 核心能力 | Spring AI ChatClient + 会话上下文管理 |
| 学段适配 | 小学生→趣味语言+引导式；中学生→知识点精讲+例题；大学生→学术讨论；成人→案例驱动 |

### 3.2 知识点讲解 📋 P0

| 项目 | 说明 |
|------|------|
| 功能描述 | 根据用户提问，检索知识库中的知识点并进行结构化讲解 |
| 计划接口 | `POST /api/tutor/explain` |
| 核心能力 | RAG (检索增强生成) + pgvector 向量检索 |
| 学段适配 | 小学生→动画/故事类比；中学生→公式推导/典型例题；成人→工作场景/实际案例 |

### 3.3 苏格拉底式引导 📋 P1

| 项目 | 说明 |
|------|------|
| 功能描述 | AI 不直接给答案，通过连续追问引导用户自己思考得出结论 |
| 计划接口 | `POST /api/tutor/guide` |
| 适用学段 | 初中、高中、大学 |

### 3.4 错题本 AI 分析 📋 P1

| 项目 | 说明 |
|------|------|
| 功能描述 | 自动收集错题，分析错误模式，定期生成"错题回顾"，推送同知识点变式题（举一反三） |
| 计划接口 | `GET /api/tutor/wrong-questions`, `POST /api/tutor/wrong-questions/analyze` |
| 适用学段 | 小学、初中、高中 |

### 3.5 个性化学习推荐 📋 P1

| 项目 | 说明 |
|------|------|
| 功能描述 | 基于学习历史和能力画像，推荐适合的学习内容和下一步学习重点 |
| 计划接口 | `POST /api/tutor/recommend` |
| 核心能力 | 用户画像 + 协同过滤 + AI 推理 |

### 3.6 多语言学习支持 📋 P2

| 项目 | 说明 |
|------|------|
| 功能描述 | 同一知识点中英文双语讲解，辅助专业英语学习 |

### 3.7 会话历史管理 📋 P2
